# 毫米波 FMCW 级联 MIMO 感知平台

这套文档用于解释一个面向毫米波 FMCW 感知任务的雷达处理与可视化项目。

仓库包含两条雷达路径：

- **TI IWR2243 Cascade**：四芯片级联雷达路径，适合更高维度的阵列感知。
- **TI IWR6843**：单芯片雷达路径，适合离线处理、热力图和点云可视化。

项目以离线流程为主。原始采集数据会被处理成距离、速度、角度和点云结果，再以图片、视频、H5 文件和 NPZ 文件的形式检查和复用。

## 这套文档怎么读

建议从简单页面开始：

1. **快速开始**：先看仓库目录和每个文件夹的作用。
2. **系统概览**：理解两条雷达硬件路径和它们的输出。
3. **处理链路**：从原始采样一步步看到可视化结果。
4. **雷达概念**：补上 FFT、虚拟阵列、AoA 和点云等核心概念。
5. **项目说明**：了解后续维护和扩展方式。

这套文档会先讲能看到什么、文件在哪里，再逐步进入 FFT、虚拟阵列、校准和点云生成。

## 输出类型

仓库会生成几类结果：

- **距离热力图**：观察目标在距离域上的响应。
- **速度热力图**：观察 Doppler 速度域上的运动响应。
- **角度热力图**：观察目标在空间角度上的响应。
- **点云**：展示三维目标结构和运动轨迹。
- **NPZ/H5 文件**：保存结构化数组，便于后续分析。
- **MP4 视频**：用于运动检查和项目展示。
