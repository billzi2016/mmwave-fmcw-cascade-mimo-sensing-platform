# 毫米波 FMCW 级联 MIMO 感知平台

这套文档用于解释一个面向毫米波 FMCW 感知任务的雷达处理与可视化项目。

仓库包含两条雷达路径：

- **TI IWR2243 Cascade**：四芯片级联雷达路径，适合更高维度的阵列感知。
- **TI IWR6843**：单芯片雷达路径，适合离线处理、热力图和点云可视化。

项目以离线流程为主。原始采集数据会被处理成距离、速度、角度和点云结果，再以图片、视频、H5 文件和 NPZ 文件的形式检查和复用。

这套代码适合用来理解从原始雷达采样到可视化结果的完整路径。它不是只调用硬件输出结果，而是把中间处理阶段保留下来：文件分组、帧重建、校准、FFT 处理、热力图生成、点云标准化和最终渲染。

## 主要工作流

把仓库拆成四条工作流，会更容易理解。

### 1. 查看演示结果

`DEMO/` 目录可以快速看到项目能生成什么结果。里面包含天线布局图、热力图视频和点云可视化结果。如果你想在读代码前先看最终效果，可以从这里开始。

### 2. 处理原始雷达采集

`Processing/` 目录读取原始雷达数据，并生成结构化输出。这里包含最重也最敏感的步骤：解析采集参数、重排原始采样、执行校准和运行 FFT。

### 3. 导出可视化资产

`Visualization/` 目录读取处理后的数组，并导出图片、视频或归一化资产。处理和展示分开后，可以在不重新跑信号处理的情况下调整显示形式。

### 4. 维护文档

`docs-site/` 目录包含这套双语文档站。英文文件放在 `docs/en/`，中文文件放在 `docs/zh/`。

## 输出类型

仓库会生成几类结果：

- **距离热力图**：观察目标在距离域上的响应。
- **速度热力图**：观察 Doppler 速度域上的运动响应。
- **角度热力图**：观察目标在空间角度上的响应。
- **点云**：展示三维目标结构和运动轨迹。
- **NPZ/H5 文件**：保存结构化数组，便于后续分析。
- **MP4 视频**：用于运动检查和项目展示。

这些输出对应不同需求。热力图紧凑，适合逐帧比较。点云更直观，适合观察空间结构。H5 和 NPZ 更适合脚本、模型训练或反复做可视化实验。

你也不需要读完整套文档才能使用代码。如果只是运行已有命令，可以先看 **快速开始**。如果要改处理逻辑，再继续看 **处理链路** 和 **雷达概念**。
